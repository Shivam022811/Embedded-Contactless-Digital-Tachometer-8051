# Testing and Verification

## Functional Verification

The embedded system was verified through simulation to evaluate firmware functionality and hardware interaction.

## Test Cases

- Sensor pulse detection
- Timer operation
- Pulse counting
- RPM calculation
- LCD output
- Continuous monitoring
- Overspeed detection
- Sensor fault indication

## Verification Outcome

The simulation verified correct pulse acquisition, accurate RPM computation, stable LCD operation, and continuous system functionality.

# Project Specification

## Project Title

**Embedded Contactless Digital Tachometer Using AT89S52 (8051)**

---

## Project Objective

To design and develop a contactless embedded system capable of measuring the rotational speed (RPM) of a rotating object using an Infrared (IR) sensor and an AT89S52 (8051) microcontroller. The system processes pulse signals in real time and displays the measured RPM on a 16×2 LCD.

---

| Programming Language    | Embedded C                      |

| Sensor                  | Infrared (IR) Sensor Module     |
| Display                 | 16×2 Character LCD              |
| Measurement Method      | Contactless Pulse Counting      |

| Power Supply            | +5 V DC                         |
| System Type             | Embedded Real-Time System       |
